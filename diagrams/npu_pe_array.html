NPU - Processing Element Array Architecture
Outer-Product Algorithm: C = A × B
For k = 0 to 3:
Broadcast A[:, k] (column k of A) → a_stage1[0..3]
Broadcast B[k, :] (row k of B) → b_stage1[0..3]
Each PE[row][col] computes:
  acc[row][col] += a_stage1[row] × b_stage1[col]
This computes: C[row][col] += A[row][k] × B[k][col]

Individual PE (Processing Element) Architecture
Multiply Unit
Input: a_value[DATA_WIDTH-1:0], b_value[DATA_WIDTH-1:0]
Output: product[2×DATA_WIDTH-1:0]
Operation: Signed multiplication
Sign Extension
Input: product[2×DATA_WIDTH-1:0]
Output: product_ext[ACC_WIDTH-1:0]
Extends sign bit to ACC_WIDTH bits
Accumulator
Register: acc_out[ACC_WIDTH-1:0]
Operation: acc_out <= acc_out + product_ext (when enable)
Clear: acc_out <= 0 (when clear)
Control Signals
clear: Reset accumulator (on start signal)
enable: valid_stage1 (enable accumulation)
clk, rst: Clock and reset
Accumulator Sizing Formula
MIN_ACC_WIDTH = (2 × DATA_WIDTH) + ceil(log2(ARRAY_SIZE))
Example (INT8, 4×4):
DATA_WIDTH = 8, ARRAY_SIZE = 4
MIN_ACC_WIDTH = 16 + 2 = 18 bits
With EXTRA_ACC_BITS = 2: ACC_WIDTH = 20 bits
